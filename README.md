
# ATOS CAPITAL SAP test

PROVA PRÁTICA FRONT-END (REACT) – 2024 


## Rodando localmente

Clone o repositório:

```bash
  git clone git@github.com:kritgarb/atos-capital.git

```
Entre na raiz do projeto:

```bash
  cd atos-capital
```

Instale as Dependências:

```bash
  npm install 
  
```
Rode a Aplicação:

```bash
   npm start
```

## Stack utilizada

**Front-end:** React
